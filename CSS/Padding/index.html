Padding

CSS tem propriedades para especificar o prenchimento de cada lado de um elemento

Esse elemento div possui Padding-top: 50px; Padding-right: 30px; Padding-bottom: 50px; Padding-left: 80px.

Prenchimento - propriedade abreviada

Para encurtar o código, é possivel especificar todas a propriedades de um prenchimento em uma propriedade

Padding-top: 25px;
Padding-right: 50px;
Padding-bottom: 75px;
Padding-left: 100px
